        # TailwindChina 工作室

        # 关于我们

        TailwindChina Studio（TailwindChina 工作室）是由几位 TailwindCSS 爱好者成立的一个工作室，宗旨是为了交流学习 TailwindCSS 以及推动 TailwindCSS 的在国内的普及。

        # 主要项目

        # TailwindCSS 中文文档

        TailwindCSS 中文文档 (opens new window) 是根据 TailwindCSS 官方英文文档翻译的简体中文版文档，目前 2.0.2 版本已经翻译完结。

        # TailwindCSS 中文网

        TailwindCSS 中文网 (opens new window) 是 TailwindChina 工作室建立的 TailwindCSS 非官方中文网站，主要包括使用教程、中文文档和相关资源。

        # TailwindCSS 生态拓展

        通过开源的方式，拓展与展示 TailwindCSS 的应用范围，目前暂定以下方向：

        • 与现有模板开源项目融合，提供基于 TailwindCSS 的实现，
        • 提供 TailwindCSS 实现的设计规范，如 Material Design, Ant Design 等。

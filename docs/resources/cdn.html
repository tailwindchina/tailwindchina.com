      # TailwindCSS CDN

      # jsDelivr

      文件 地址
      base.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/base.css (opens new window)
      base.min.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/base.min.css (opens new window)
      components.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/components.css (opens new window)
      components.min.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/components.min.css (opens new window)
      tailwind-dark.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind-dark.css (opens new window)
      tailwind-dark.min.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind-dark.min.css (opens new window)
      tailwind-experimental.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind-experimental.css (opens new window)
      tailwind-experimental.min.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind-experimental.min.css (opens new window)
      tailwind.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind.css (opens new window)
      tailwind.min.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind.min.css (opens new window)
      utilities.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/utilities.css (opens new window)
      utilities.min.css https://cdn.jsdelivr.net/npm/tailwindcss/dist/utilities.min.css (opens new window)

      注意

      默认所有地址为已发布的最新版本。

      如果要指定具体版本号，请在 tailwindcss 后面加上 @版本号。

      例如，我想使用 2.0.2 版本，那么CDN地址就是 https://cdn.jsdelivr.net/npm/tailwindcss@2.0.2/dist/tailwind.css
